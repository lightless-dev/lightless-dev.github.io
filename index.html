lightLess: A journey through a colorless world

lightLess is a fun parkour game that can be played on your browser-no installation needed! Jump and run through the obstacles... the levels are short and fun!!!

Description

lightLess is an web-based, parkour based game with short and fun levels. navigate through by using the arrow keys or WASD keys. Have fun as you beat the levels - You'll need some practice! lightLess was built with Scratch, converted to HTML and posted on the web with github pages. Thank you to all our developers and you for playing. please, leave us feedback.
